Integration
Options

For uGet: [URL]
Defines how the link to be downloaded is provided to the download manager.
Quiet Mode: --quiet
Skips download dialog, starts the download immediately.

Download Direct to a Folder: --folder [FOLDER]
Add a value similar to "/home/user/Downloads" (without quotes) in the Destination field.
